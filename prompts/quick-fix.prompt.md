# Quick Fix

No analysis paralysis. Paste error → get fix → move on.

## Input
Error message or one-line description of the issue.

## Rules
1. **Lead with the fix** — show code first, explain after
2. **Max 3 sentences** explanation
3. If multiple possible causes, show the most likely one first
4. Include the command to verify the fix
5. If you need more context, ask ONE specific question

## Template

```
## Fix
[code block with the fix]

## Why
[1-2 sentences]

## Verify
[command to run]
```
